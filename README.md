# Tapioca zeus_wrapper
The Zeus-wrapper is a library for easy use of the Zeus API. Zeus is a unique platform for communicating with Intelbras devices.


## Installation
```
pip install tapioca-zeus_wrapper
```

## Documentation
``` python
from tapioca_zeus_wrapper import Zeus_wrapper


api = Zeus_wrapper()

```
